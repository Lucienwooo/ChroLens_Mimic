## 功能介紹：</br>
1.巨集錄製工具：可以錄製鍵盤和滑鼠的所有操作，並將其儲存為巨集。</br>
2.設定重複的次數或是指定的時間範圍進行重複</br>
3.可自定義快捷鍵、腳本名稱、Skin</br>

### 情境範例：</br>
1.數筆excel表格內容複製到資料庫貼上</br>
2.滑鼠左右滑動，預防電腦進入待機或登入畫面</br>
3.遊戲重複操作：定點採集/簡單左右移動/重複施放技能</br>
