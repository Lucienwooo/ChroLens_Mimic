# 🎮 智能戰鬥系統 - 快速參考

## 🚀 立即開始

### 從主程式啟動 (推薦)
```
1. 運行 ChroLens_Mimic.py
2. 點擊左側選單 "4.自動戰鬥"
3. 開啟智能戰鬥視窗
```

### 獨立啟動
```bash
python 啟動智能戰鬥.py
```

---

## ⚡ 3 步驟快速設定

### 1️⃣ 設定視窗
- 輸入遊戲視窗標題
- 點擊「測試連接」驗證

### 2️⃣ 截取圖片
- 點「📸 截取角色」(3秒倒數)
- 點「📸 截取敵人」(可多個)

### 3️⃣ 開始運行
- 調整參數 (可選)
- 點「▶️ 開始運行」

---

## 📊 核心功能

✅ 自適應地圖學習
✅ 智能敵人偵測
✅ 自動戰鬥攻擊
✅ 卡住自動脫困
✅ 血量自動補給
✅ 即時統計追蹤

---

## ⚙️ 重要參數

| 參數 | 建議值 | 說明 |
|------|--------|------|
| 攻擊範圍 | 100-150px | 超過距離會先移動 |
| 補血閾值 | 50-60% | 低於此值自動喝水 |
| 探索時長 | 5-10分鐘 | 首次建議短時間測試 |

---

## 🎯 預設按鍵

- **跳躍**: Alt
- **攻擊**: Ctrl
- **技能**: A, S, D
- **補血**: Page Up
- **補魔**: Page Down

💡 根據遊戲調整!

## 🐛 快速排錯

### 找不到視窗?
→ 檢查視窗標題是否正確

### 找不到角色?
→ 重新截取更清晰的圖片

### 不會攻擊?
→ 檢查「自動攻擊」是否啟用

### 一直卡住?
→ 系統會自動脫困,選簡單地圖
