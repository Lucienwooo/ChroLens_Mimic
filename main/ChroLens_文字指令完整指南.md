## 🚀 快速開始

### 預設腳本格式
每個新腳本都會包含以下標題：
```
# ChroLens 文字指令腳本
# 預設按鍵持續時間: 50ms
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

### 基本工作流程
1. **錄製基本操作** → 使用主程式錄製功能
2. **開啟編輯器** → 編輯腳本
3. **加入圖片辨識** → 使用「📷 圖片辨識」截圖
4. **加入條件判斷** → 使用 `>if>圖片`
5. **加入循環** → 使用標籤和跳轉
6. **儲存並測試** → 按「💾 儲存」→「▶️ 執行」

## 🎯 基本指令語法

### ⌨️ 鍵盤操作

#### 1. 按鍵指令
```
>按KEY, 延遲Xms, T=時間
```

**參數說明**：
- `KEY`：按鍵名稱（如：A, B, Enter, Space, Ctrl, Shift）
- `延遲Xms`：按鍵持續時間（毫秒）
- `T=時間`：指令執行的時間點（格式：XsXXX）

**範例**：
```
>按A, 延遲50ms, T=0s000          # 按下並放開
>按Enter, 延遲50ms, T=0s100       # Enter鍵
>按Space, 延遲50ms, T=0s200       # 空白鍵
>按下Ctrl, 延遲0ms, T=0s300       # 只按下（不放開）
>放開Ctrl, 延遲0ms, T=0s400       # 只放開
```

#### 2. 按下按鍵（不放開）
```
>按下KEY, T=時間
```

#### 3. 放開按鍵
```
>放開KEY, T=時間
```

**組合鍵範例**（Ctrl+C）：
```
>按下Ctrl, 延遲0ms, T=0s000
>按C, 延遲100ms, T=0s000
>放開Ctrl, 延遲0ms, T=0s100
```

---

### 🖱️ 滑鼠操作

#### 1. 移動滑鼠
```
>移動至(X,Y), T=時間
```

**範例**：
```
>移動至(100,200), T=0s000         # 移動游標
>移動至(500,300), T=1s000
```

#### 2. 滑鼠點擊
```
>左鍵點擊(X,Y), T=時間
>右鍵點擊(X,Y), T=時間
>中鍵點擊(X,Y), T=時間
```

**範例**：
```
>左鍵點擊(100,200), T=0s100       # 左鍵點擊
>右鍵點擊(100,200), T=0s200       # 右鍵點擊
```

#### 3. 滑鼠按下/放開（拖曳）
```
>按下left鍵(X,Y), T=時間
>放開left鍵(X,Y), T=時間
```

**拖曳範例**：
```
>移動至(100,100), T=0s000
>按下left鍵(100,100), T=0s100     # 按下不放
>移動至(500,500), T=0s500
>放開left鍵(500,500), T=0s600     # 放開（完成拖曳）
```

## 🖼️ 圖片辨識指令

### 基本圖片操作

#### 1. 辨識圖片
```
>辨識>圖片名稱, T=時間
```

#### 2. 移動到圖片位置
```
>移動至>圖片名稱, T=時間
```

#### 3. 點擊圖片
```
>左鍵點擊>圖片名稱, T=時間
>右鍵點擊>圖片名稱, T=時間
```

**範例**：
```
>辨識>pic01, T=0s000              # 辨識圖片
>移動至>pic01, T=0s100            # 移動到圖片位置
>左鍵點擊>pic01, T=0s200          # 點擊圖片（會回原位）
>右鍵點擊>pic02, T=0s300          # 右鍵點擊圖片
```

#### 4. 多圖辨識（找到任一張）
```
>辨識任一>pic01|pic02|pic03, T=時間
```

**範例**：
```
>辨識任一>pic01|pic02, T=0s400   # 多圖辨識（找到任一）
```

### 🔀 條件判斷

#### 基本條件語法
```
>if>圖片名稱, T=時間
>>#成功標籤                         # 找到 → 跳到 #成功標籤
>>>#失敗標籤                        # 找不到 → 跳到 #失敗標籤
```

**範例**：
```
>if>pic01, T=0s000                # 檢查圖片是否存在
>>#找到了                          # 成功分支
>>>#沒找到                          # 失敗分支
```

#### 簡化跳轉語法
```
>>#標籤名稱        # 簡化格式（推薦）
>>跳到#標籤名稱    # 完整格式（兩者相同）
```

---

### 🏷️ 標籤與跳轉

#### 1. 定義標籤
```
#標籤名稱
```

#### 2. 跳轉到標籤
```
>跳到#標籤名稱, T=時間
```

**範例**：
```
#開始                             # 定義標籤
>左鍵點擊(100,100), T=0s000
>跳到#開始, T=1s000               # 無條件跳轉（無限循環）
```

---

### 🔄 循環邏輯

#### 1. 無限循環
```
#開始
>左鍵點擊(100,100), T=0s000
>跳到#開始, T=0s100
```

#### 2. 條件循環（找到圖片為止）
```
#尋找
>if>目標圖片, T=0s000
>>#找到了
>>>#繼續找

#繼續找
>跳到#尋找, T=0s500

#找到了
>左鍵點擊>目標圖片, T=1s000
```

#### 3. 複雜循環（遊戲腳本）
```
#找
>if>pic04, T=0s100
>>#點擊
>>>#找

#點擊
>移動至>pic04, T=0s500
>左鍵點擊>pic04, T=0s600
>跳到#找, T=1s000
```

### 案例 1：自動打怪系統

**場景**：RPG遊戲自動刷怪

**功能**：
- 偵測怪物出現
- 追逐怪物
- 施放技能攻擊
- 撿取掉落物
- 自動補血

```

#開始巡邏
>移動至(800,600), T=0s000
>左鍵點擊(800,600), T=0s100

#尋找怪物
>if>怪物頭像, T=0s500
>>#攻擊怪物
>>>#檢查血量

#攻擊怪物
>左鍵點擊>怪物頭像, T=1s000
>按1, 延遲100ms, T=1s200
>按2, 延遲100ms, T=1s500
>按3, 延遲100ms, T=1s800
>if>怪物死亡特效, T=2s500
>>#撿取掉落物
>>>#繼續攻擊

#繼續攻擊
>按1, 延遲100ms, T=3s000
>跳到#攻擊怪物, T=3s200

#撿取掉落物
>if>掉落物圖標, T=3s500
>>#撿取
>>>#尋找怪物

#撿取
>左鍵點擊>掉落物圖標, T=4s000
```

1. 移動到巡邏點

7. 定期檢查血量，血量低就補血

## 🚀 圖片辨識速度優化

### ❌ 原始問題
腳本執行時間遠超預期：
```
#找
>if>pic04, T=0s000
>>#點擊
>>>#找
#點擊
>移動至>pic04, T=0s048
>辨識>pic04, T=0s098
>左鍵點擊>pic04, T=0s348
```

**理論執行時間**：0.348秒  
**實際執行時間**：16秒以上（每次圖片辨識 3-5秒）

---

### ✅ 優化方案

#### 舊版本（慢速精確模式）
- 🔍 測試 **9種不同尺度** (0.8 ~ 1.2)
- 🔍 使用 **3種匹配方法** 並加權平均
- 🔍 進行 **SSIM結構相似度** 驗證
- 🔍 進行 **直方圖** 驗證
- 🔍 進行 **邊緣檢測** 驗證
- ⏱️ **每次辨識耗時**：3-5秒

#### 新版本（快速模式）
- ✅ 只測試 **3種關鍵尺度** (0.9, 1.0, 1.1)
- ✅ 只使用 **單一最佳匹配方法**
- ✅ **跳過所有驗證步驟**
- ✅ 降低預設閾值：0.92 → 0.75
- ⏱️ **每次辨識耗時**：0.3-0.5秒

### 💡 提升辨識準確度的技巧

- ✅ 圖片要清晰，避免模糊

- ✅ 太小：20x20 像素以上
- ✅ 適中：50x50 到 200x200 像素

#### 4. 背景對比
- ✅ 截取包含周圍背景的圖片
- ✅ 確保目標與背景有明顯對比
- ❌ 避免純色背景 + 純色圖標

## 📏 時間格式說明

```
T=0s000     → 0秒
T=0s100     → 0.1秒
T=0s500     → 0.5秒
T=1s000     → 1秒
T=5s500     → 5.5秒
T=1m30s000  → 1分30秒
```

格式：`T=XmYsZZZ`
- `X`：分鐘數
- `Y`：秒數
- `ZZZ`：毫秒數

## ⚠️ 常見錯誤與解決方案

### 錯誤 1：標籤不存在
```
>if>pic01, T=0s000
>>#不存在的標籤    # ← 找不到這個標籤！
```

**解決**：確保標籤已定義
```
#不存在的標籤
>左鍵點擊(100,100), T=0s000
```

---

### 錯誤 2：圖片檔案不存在
```
>左鍵點擊>pic99, T=0s000    # ← pic99.png 不存在！
```

**解決**：使用編輯器的「📷 圖片辨識」功能截圖

---

### 錯誤 3：無限循環沒有跳出條件
```
#死循環
>左鍵點擊(100,100), T=0s000
>跳到#死循環, T=0s100         # ← 永遠不會停止！
```

**解決**：加入條件判斷
```
#循環
>if>停止標記, T=0s000
>>#結束
>>>#繼續

#繼續
>左鍵點擊(100,100), T=0s100
>跳到#循環, T=0s200

#結束
```

---

### 錯誤 4：圖片辨識失敗
**原因**：
- 圖片模糊
- 背景變化
- 尺寸不匹配

**解決**：
1. 重新截取清晰的圖片
2. 包含足夠的周圍背景
3. 確保對比度足夠

---

### 錯誤 5：時間間隔太短
```
>左鍵點擊(100,100), T=0s000
>左鍵點擊(200,200), T=0s001  # ← 間隔太短！
```

**解決**：增加時間間隔
```
>左鍵點擊(100,100), T=0s000
>左鍵點擊(200,200), T=0s100  # ← 至少間隔 100ms
```
