# ChroLens_Mimic 打包說明

## 快速打包

```bash
python pack.py
```

## 打包流程

1. **清理舊檔案** - 刪除 build/、dist/、*.spec
2. **PyInstaller 打包** - 使用 `--noconsole --onedir` 參數
3. **創建 ZIP** - 自動壓縮為 `ChroLens_Mimic_{版本}.zip`
4. **清理建置檔案** - 保留 dist/ 中的 ZIP，刪除其他檔案

## 輸出位置

- **exe 程式**: `dist/ChroLens_Mimic/ChroLens_Mimic.exe`
- **ZIP 檔案**: `dist/ChroLens_Mimic_{版本}.zip`

## 注意事項

- 確保 `pic/umi_奶茶色.ico` 存在
- Python 3.8+ 環境
- 需要安裝 PyInstaller: `pip install pyinstaller`

## 版本管理

版本號從 `ChroLens_Mimic.py` 中的 `VERSION` 變數讀取。
